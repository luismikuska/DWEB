Hyperlink ou link

O que é hiperlink?

É a referência dentro de um documento em hipertexto para outras partes do mesmo documento,
outros documentos, arquivos ou endereço eletrônico.

Tipos de links

  • Remotos
  • Internos
  • Locais

Hiperlinks externos ou remotos

São LINKS que referenciam sites ou página da web.
Na propriedade href utilizamos a URL (Uniform Resource Lacator) para a referência que desejamos.

Hiperlinks Internos

São links que apontam a um lugar diferente dentro da mesma página e
possibilita acessar diferentes parágrafo ou sessões quando o documento é extenso.

      Podem ser acessados :

    1. áudio
    2. texto
    3. vídeo
    4. imagem

Links internos são considerados links em seu site que apontam para outro artigo do seu próprio site.

Hiperlinks locais

São links que apontam para outra página dentro de um mesmo site.

Hiperlinks em imagens

É possível realizar hiperlinks e imagens envolvendo
a tag de imagem com a tag de hiperlink

Hiperlinks para downloads de arquivos

São links que apontam para arquivos e possíbilitamrealizar download destes arquivos.
