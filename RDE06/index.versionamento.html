Software de versionamento

O que é versionamento?

Versionamento de software é um processo de controle de versões
estabelecidos por meio de numerações diferentes. isso permite que os programadores
saibam quando e quais alterações foram realizadas, acompanhando as mudanças
criadas no software , também permitem que os usuários finais identifiquem as novidades
e reconheçam as versões mais atualizadas.

Software de controle de versões

O software de controle de versão é um sistema ou VCS ou sistema de controle de fonte,
é um recurso que monitora e gerencia as mudanças em um sistema de arquivos. Um VCS
tambem oferece recursos colaborativos para compartilhar e integrar essas mudanças de
sistema de arquivos com outros usuários do VCS.

Git

É um software de versionamento que surgiu para o desenvolvimento do Kernel do Linux,
foi projetado e desenvolvido por Linux Torvalds em 2005. Um repositório com histórico completo
de acompanhamento de revisões, não depende de acesso a uma rede ou a um centralservidor .

GitHub

Platafrma de hospedagem de código-fonte e arquivos com controle de vesão,surgiu em 2008 e
foi adquirido em 2018 pela Microsoft por US$ 7,5 bilhões, atualmente ele hospeda 100 milhões de
projetos com 56 milhões de usuários.
